# 🎛️ KRONOS AI NEXUS — Экосистема Локального ИИ и Системных Модулей v1.0

**KRONOS AI NEXUS** — это модульный ИИ-Хаб и консольная операционная среда, разработанная на Python. Проект спроектирован для бесшовного запуска локальных нейросетей (on-device inference), глубокого анализа текстовых массивов, автоматической верификации кода и интеллектуального ассистирования в режиме реального времени.

Вся экосистема построена на принципах строгой модульной изоляции: каждый активный инструмент монолитно занимает одну сессию терминала, эффективно очищая кэш ОЗУ при завершении работы.

---

## 🧠 Ключевая Архитектура и Модули

Система управляется главным ядром `KRONOS_NEXUS.py`, которое координирует работу пяти независимых подмодулей, расположенных в директории `modules/`:

1. **🤖 KRONOS AI PROTOTYPE (`modules/kronos_chat.py`)**
   Интерактивный ИИ-чат-терминал. Оснащен долгосрочной локальной JSON-памятью контекста, алгоритмами подсчета времени генерации токенов и очистки буфера.

3. **🌀 DATA VORTEX PARSER (`modules/kronos_vortex.py`)**
   Высокопроизводительный посекторный анализатор текстовых массивов. Нарезает неструктурированные данные из `vortex_input.txt` на перекрывающиеся чанки (chunks), отправляет их на конвейер локального ИИ и собирает хронологический "Гранд-Отчет".

4. **📐 SMART SOLVER & CODE EXECUTOR (`modules/kronos_solver.py`)**
   Инженерный вычислитель с функцией авто-верификации. Локально компилирует и тестирует сгенерированный Python-код непосредственно на процессоре (CPU Executor), перехватывая ошибки синтаксиса и логики.

5. **🧠 DYNAMIC AI MENTOR (`modules/kronos_mentor.py`)**
   Гибридный модуль кастомизации личности ИИ. Позволяет пользователю на лету задавать системный промпт (например, "Жесткий тренер по бодибилдингу" или "Дипломат WeChat"), совмещая текстовые ответы нейросети с точными локальными расчетами нагрузок на чистом Python.

6. **🛡️ OS COMMANDER MODE (`modules/kronos_commander.py`)**
   Интерфейс низкоуровневой автоматизации для прямого безопасного управления процессами и конфигурациями операционной системы Windows.

## 🛠️ Технологический Стек

* **Язык разработки:** Python 3.12+
* **Интерфейсный движок:** `rich` (полная поддержка градиентов, панелей, таблиц и анимированных статус-баров в Windows Terminal).
* **Локальное ИИ-ядро:** `ollama` (базовая конфигурация под инференс моделей `qwen2.5:3b`, `deepseek-r1:8b` или `llama3`).
* **Системные библиотеки:** `subprocess`, `sys`, `os`, `json`, `math`, `time`.
